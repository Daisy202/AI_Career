For my final year project when I'm designing the system, which pages should l use or what sections do you think we should use

Since your project is “An AI-Driven Framework for Personalized Career and Academic Guidance for Pre-University Students in Zimbabwe”, the system you design will likely be a web or mobile platform where students interact with the AI to get career recommendations.

Below are recommended system pages/sections you can include in your system design. These are realistic and suitable for an AI career guidance platform.

1. Landing Page / Home Page
This is the first page users see.

Main content:

Brief introduction of the platform

Explanation of how the system helps students choose careers

Buttons for:

Sign Up

Login

Learn More

Overview of features

2. User Registration Page
Allows new students to create accounts.

Fields may include:

Full Name

Email Address

Password

School Name

Level (Form 4 / Form 6)

Accept Terms and Conditions

3. Login Page
Allows registered users to access the system.

Fields:

Email / Username

Password

Forgot Password option

4. Student Profile Page
This page stores personal and academic information.

Information collected:

Personal details

Subjects taken at school

Academic performance

Interests

Skills

Personality traits

This data helps the AI engine make recommendations.

5. Career Assessment / Questionnaire Page
This is one of the core pages of your system.

Students answer questions such as:

What subjects do you enjoy most?

What type of work interests you?

Do you prefer working with people, technology, or data?

What are your hobbies?

The answers feed the AI recommendation model.

6. AI Career Recommendation Page
This page displays career suggestions generated by the AI system.

Example output:

Recommended careers

Matching percentage

Why the career suits the student

Required subjects

Suggested university programs

Example:

Career	Match %
Software Engineer	92%
Data Analyst	85%
Network Engineer	80%
7. Career Information Page
Provides detailed information about each career.

Information may include:

Career description

Required skills

Required subjects

University programs in Zimbabwe

Possible job opportunities

8. University / Program Suggestions Page
Shows universities and programs related to the recommended career.

Example:

Universities offering the program

Entry requirements

Duration of study

Example institutions:

Bindura University

University of Zimbabwe

Chinhoyi University of Technology

9. Dashboard
The main page after login.

Shows:

Student progress

Completed assessments

Recommended careers

Saved careers

Notifications

10. Admin Panel
Used by system administrators.

Functions may include:

Manage students

Update career information

Monitor system performance

Update AI model or datasets

11. Feedback Page
Students can:

Rate recommendations

Report issues

Suggest improvements

This helps improve the AI model accuracy.

12. Help / Support Page
Provides:

FAQs

Contact information

Guidance on how to use the system

Simple System Flow
A typical user journey could be:

Home Page

Register / Login

Complete Profile

Take Career Assessment

AI Generates Recommendations

View Career Details

Explore University Programs
